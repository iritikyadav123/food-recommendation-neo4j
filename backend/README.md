### Quick start guide

```bash
yarn install


# create the .env file and add

PORT=4000
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=Ritik@123



yarn start
```

The username and password for the default provided dataset is neo4j/admin.
